# 🚀 Nexus AI — Smart Study Planner

Nexus AI is an AI-powered study planning system that generates, refines, and optimizes study schedules in real time.

---

## ✨ Features

* ⚡ AI-generated study plans using Gemini API
* 🔁 Real-time refinement (e.g. “Make it harder”)
* ⏱️ Built-in Pomodoro focus timer
* 🎯 Fixed total time constraint system
* 🎨 Modern glassmorphism UI (fully responsive)
* 🧠 Subject-based customization with difficulty levels

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* Google Gemini API

## ⚙️ How It Works

1. Enter your Gemini API key
2. Add subjects and context
3. Generate a study plan
4. Refine it using custom instructions
5. Use the built-in timer to execute
